## 哪些常用的激活函数？  
在神经网络中，激活函数用于引入非线性因素，使得模型能够学习和表示更复杂的函数。以下是一些常用的激活函数及其特点：  
  
1. **Sigmoid**  
    - 公式：$$ \sigma(x) = \frac{1}{1 + e^{-x}} $$  
    - 特点：输出在 `(0, 1)` 之间，适合二分类问题的输出层。  
    - 缺点：容易导致梯度消失  
  
2. **ReLU**  
    - 公式：$$ \text{ReLU}(x) = \max(0, x) $$  
    - 特点：计算简单，缓解梯度消失问题，是目前深度学习中最常用的激活函数之一。  
    - 缺点：对于负数输入，可能导致神经元“死亡”  
  
3. **Softmax**  
    - 公式：$$ \text{Softmax}(x_i) = \frac{e^{x_i}}{\sum_{j} e^{x_j}}, \quad \text{其中 } i, j \text{ 为输入向量中的元素索引} $$  
    - 特点：将输入转换为概率分布，常用于多分类问题的输出层。  
  
通常，ReLU 及其变种适用于隐藏层，而 Sigmoid 和 Softmax 常用于输出层。

## NumPy 数组与 PyTorch 张量的关系

### 相同点
1. **都是多维数组**
   - 都用来存储和操作数值数据
   - 都支持向量化运算（不用写循环）
   - 都有形状（shape）、数据类型（dtype）等属性

2. **API 相似**
   ```python
   # NumPy
   a = np.array([1, 2, 3])
   b = a + 10
   
   # PyTorch
   a = torch.tensor([1, 2, 3])
   b = a + 10
   ```

### 关键区别

| 特性 | NumPy 数组 | PyTorch 张量 |
|------|-----------|-------------|
| 自动求导 | 不支持 | 支持（`requires_grad=True`） |
| GPU 加速 | 不支持 | 支持（`.to('cuda')`） |
| 用途 | 通用科学计算 | 深度学习专用 |
| 梯度追踪 | 无 | 有计算图 |

### 转换关系

```python
# NumPy → Tensor
x_np = np.array([1, 2, 3])
x_tensor = torch.tensor(x_np)  # 或 torch.from_numpy(x_np)

# Tensor → NumPy
x_tensor = torch.tensor([1, 2, 3])
x_np = x_tensor.numpy()
```

### 为什么深度学习用张量？

```python
w = torch.nn.Parameter(torch.rand(1), requires_grad=True)
```
- `requires_grad=True` 让 PyTorch 自动记录对 `w` 的所有操作
- 调用 `loss.backward()` 时，自动计算 `w` 的梯度
- 如果用 NumPy，需要手动写求导公式

**简单说**：张量 = NumPy 数组 + 自动求导 + GPU 支持


```python
import torch
import numpy as np
import matplotlib.pyplot as plt
from torch.utils.data import TensorDataset, DataLoader

# 数据准备：生成"凸起"形状的数据
# x 值从 -10 到 10，对应的 y 值呈现"山峰"形状（中间是 1，两边是 0）
x = np.random.randint(-10, 10, 150)
x.sort()
y0 = np.zeros(50)   # 前 50 个点标签为 0
y1 = np.ones(50)    # 中间 50 个点标签为 1
y2 = np.zeros(50)   # 后 50 个点标签为 0
y = np.concatenate((y0, y1, y2))  # 合并成 [0,0,...,1,1,...,0,0,...]

# 将 NumPy 数组转换为 PyTorch 张量
x_tensor = torch.tensor(x, dtype=torch.float32)
y_tensor = torch.tensor(y, dtype=torch.float32)

# 创建数据集和数据加载器
dataset = TensorDataset(x_tensor, y_tensor)
dataloader = DataLoader(dataset, batch_size=5, shuffle=True)

# 定义网络参数（两层神经网络）
# 隐藏层：2 个 Sigmoid 神经元，各有自己的权重和偏置
w1 = torch.nn.Parameter(torch.rand(1), requires_grad=True)  # 第一个 Sigmoid 的权重
b1 = torch.nn.Parameter(torch.rand(1), requires_grad=True)  # 第一个 Sigmoid 的偏置
w2 = torch.nn.Parameter(torch.rand(1), requires_grad=True)  # 第二个 Sigmoid 的权重
b2 = torch.nn.Parameter(torch.rand(1), requires_grad=True)  # 第二个 Sigmoid 的偏置

# 输出层：线性组合两个 Sigmoid 的输出
w3 = torch.nn.Parameter(torch.rand(1), requires_grad=True)  # 第一个 Sigmoid 的组合权重
w4 = torch.nn.Parameter(torch.rand(1), requires_grad=True)  # 第二个 Sigmoid 的组合权重
b3 = torch.nn.Parameter(torch.rand(1), requires_grad=True)  # 输出层偏置

# 优化器需要同时调整 7 个参数
optimizer = torch.optim.SGD([w1, w2, w3, w4, b1, b2, b3], lr=0.01)
criterion = torch.nn.MSELoss()  # 均方误差损失


def sigmoid(x):
    return 1 / (1 + torch.exp(-x))


# 训练循环
epochs = 1000
for epoch in range(epochs):
    for x_batch, y_batch in dataloader:
        # 隐藏层：两个 Sigmoid 神经元
        s1 = sigmoid(w1 * x_batch + b1)  # 第一个神经元（负责"上升"部分）
        s2 = sigmoid(w2 * x_batch + b2)  # 第二个神经元（负责"下降"部分）
        
        # 输出层：线性组合两个 Sigmoid 的输出
        y_predict = w3 * s1 + w4 * s2 + b3
        
        # 计算损失并更新参数
        loss = criterion(y_predict, y_batch)
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()

# 最终预测结果
sigmoid1 = sigmoid(w1 * x_tensor + b1)
sigmoid2 = sigmoid(w2 * x_tensor + b2)
final_prediction = w3 * sigmoid1 + w4 * sigmoid2 + b3

# 绘图
plt.scatter(x, y)
# detach() 的作用：从计算图中分离张量，避免在绘图时触发梯度计算
plt.plot(x, final_prediction.detach().numpy(), color='r')
plt.show()
```

## 两层神经网络的工作原理

### 网络结构

```
输入 x → [Sigmoid 1] → s1 ↘
                              → w3*s1 + w4*s2 + b3 → 输出
输入 x → [Sigmoid 2] → s2 ↗
```

### 为什么需要两个 Sigmoid？

单个 Sigmoid 只能拟合单调的 S 型曲线，无法表示"先上升后下降"的凸起形状。

两个 Sigmoid 组合可以：
- s1 负责"上升"部分（左边从 0 到 1）
- s2 负责"下降"部分（右边从 1 到 0）
- 通过 w3、w4、b3 调整它们的权重和偏移，合成凸起形状

### 核心思想:
- 单个神经元能力有限（单个 Sigmoid 只能拟合单调曲线）
- 多个神经元组合可以拟合复杂模式（两个 Sigmoid 组合拟合凸起）
- 这就是为什么深度学习需要"多层"和"多神经元"

## 前向传播与反向传播详解

### 什么是前向传播（Forward Propagation）

**定义**：数据从输入层经过网络各层，最终得到输出的过程。

**过程示意**：
```
输入 x → 隐藏层计算 → 输出层计算 → 得到预测值 y_predict
```

**代码示例**：
```python
# 两层神经网络的前向传播
s1 = sigmoid(w1 * x + b1)           # 隐藏层第一个神经元
s2 = sigmoid(w2 * x + b2)           # 隐藏层第二个神经元
y_predict = w3 * s1 + w4 * s2 + b3  # 输出层，得到预测值
```

**作用**：根据当前的权重和偏置参数，计算出模型的预测结果。

---

### 什么是反向传播（Backward Propagation）

**定义**：根据损失函数，从输出层向输入层反向计算每个参数的梯度（导数）的过程。

**过程示意**：
```
损失 loss ← 输出层梯度 ← 隐藏层梯度 ← 计算每个参数的梯度
```

**代码示例**：
```python
loss = criterion(y_predict, y_true)  # 1. 计算损失（预测值和真实值的差距）
optimizer.zero_grad()                # 2. 清空上一轮的梯度
loss.backward()                      # 3. 反向传播，自动计算所有参数的梯度
optimizer.step()                     # 4. 根据梯度更新参数
```

**作用**：告诉每个参数应该往哪个方向调整、调整多少，才能减小损失。


### 梯度是什么？

**梯度**就是函数的导数，表示"如果我改变这个参数，损失会如何变化"。

```python
# 假设损失函数 loss = (y_predict - y_true)^2
# 如果 w 的梯度是 +2，意味着：
#   - w 增加 → loss 增加
#   - w 减少 → loss 减少
# 所以应该让 w 减小

# 如果 w 的梯度是 -3，意味着：
#   - w 增加 → loss 减少
#   - w 减少 → loss 增加
# 所以应该让 w 增大
```

**参数更新公式**：
```python
w_new = w_old - learning_rate * gradient
# learning_rate（学习率）控制每次调整的步长
```


### 完整训练流程

```python
for epoch in range(epochs):
    for x_batch, y_batch in dataloader:
        # ========== 前向传播 ==========
        # 1. 输入数据通过网络，计算预测值
        y_predict = model(x_batch)
        
        # 2. 计算损失（预测值与真实值的差距）
        loss = criterion(y_predict, y_batch)
        
        # ========== 反向传播 ==========
        # 3. 清空上一轮的梯度（必须执行，否则梯度会累加）
        optimizer.zero_grad()
        
        # 4. 反向传播：自动计算所有参数的梯度
        #    PyTorch 会自动应用链式法则，计算 loss 对每个参数的导数
        loss.backward()
        
        # 5. 更新参数：根据梯度调整权重和偏置
        #    w = w - learning_rate * gradient
        optimizer.step()
```


### 形象比喻：爬山找最低点

假设你在山上，想找到最低点（最小化损失）：

| 步骤 | 神经网络 | 爬山比喻 |
|------|---------|---------|
| **前向传播** | 输入数据 → 计算输出 | 从当前位置往前走一步 |
| **计算损失** | 对比预测值和真实值 | 测量当前位置的海拔高度 |
| **反向传播** | 计算梯度 | 用指南针判断哪个方向最陡（下坡最快） |
| **更新参数** | 调整权重和偏置 | 朝着最陡的下坡方向走一步 |

**学习率**就像步长：
- 学习率太大：步子太大，可能跳过最低点
- 学习率太小：步子太小，走得很慢
- 合适的学习率：稳定地接近最低点


### 为什么需要反向传播？

**手动计算梯度的困难**：
```python
# 假设网络有 3 层，损失函数是 MSE
# 需要手动求导（链式法则）：
dL/dw1 = dL/dy * dy/ds1 * ds1/dw1  # 复杂的链式求导
dL/dw2 = dL/dy * dy/ds2 * ds2/dw2
dL/db1 = dL/dy * dy/ds1 * ds1/db1
# ... 网络越深，公式越复杂，容易出错
```

**使用反向传播的优势**：
```python
loss.backward()  # PyTorch 自动完成所有求导，无论网络有多深
```

**核心优势**：
1. **自动化**：无需手动推导复杂的求导公式
2. **准确性**：避免人工计算错误
3. **高效性**：优化的计算图，速度快
4. **可扩展**：无论网络有多少层、多少参数，都能自动处理


### 为什么要 optimizer.zero_grad()？

```python
# 第一次迭代
loss.backward()  # w.grad = 2.5

# 第二次迭代（如果不清空梯度）
loss.backward()  # w.grad = 2.5 + 3.1 = 5.6  ❌ 错误！梯度累加了

# 正确做法
optimizer.zero_grad()  # w.grad = 0
loss.backward()        # w.grad = 3.1  ✓ 正确
```

**原因**：PyTorch 默认会累加梯度（为了支持某些高级用法），所以每次反向传播前必须手动清零。


### 实际例子：单个参数的更新过程

```python
# 假设当前参数 w = 0.5，学习率 lr = 0.01
# 第一次迭代
y_predict = w * x          # 前向传播
loss = (y_predict - y)^2   # 计算损失
loss.backward()            # 反向传播，得到 w.grad = -1.2
w = w - lr * w.grad        # 更新：w = 0.5 - 0.01 * (-1.2) = 0.512

# 第二次迭代
optimizer.zero_grad()      # 清空梯度
y_predict = w * x          # 前向传播（使用新的 w = 0.512）
loss = (y_predict - y)^2   # 计算损失
loss.backward()            # 反向传播，得到 w.grad = -0.9
w = w - lr * w.grad        # 更新：w = 0.512 - 0.01 * (-0.9) = 0.521

# ... 重复多次，w 逐渐接近最优值
```


| 概念 | 作用 | 方向 |
|------|------|------|
| **前向传播** | 计算预测值 | 输入 → 输出 |
| **损失函数** | 衡量预测误差 | - |
| **反向传播** | 计算梯度 | 输出 → 输入 |
| **参数更新** | 优化参数 | 梯度下降 |

**记忆口诀**：
- 前向算预测，反向算梯度
- 梯度指方向，学习率定步幅
- 清零防累加，更新向最优

```python
import torch  
import numpy as np  
import matplotlib.pyplot as plt  
from torch.utils.data import TensorDataset, DataLoader  
from torch.utils.tensorboard import SummaryWriter  
  
# 数据准备  
x = np.random.randint(80, 100, 200)  
x.sort()  
y0 = np.zeros(50)  
y1 = np.ones(50)  
y2 = np.zeros(50)  
y3 = np.ones(50)  
y = np.concatenate((y0, y1, y2, y3))  # 合并两个数组  
  
# [1,2]  -> (2, 1)  
# 将 NumPy 数组转换为 PyTorch 张量 (1, 200) -> (200, 1)x_tensor = torch.tensor(x, dtype=torch.float32).view(-1, 1)  
y_tensor = torch.tensor(y, dtype=torch.float32).view(-1, 1)  
  
dataset = TensorDataset(x_tensor, y_tensor)  
dataloader = DataLoader(dataset, batch_size=5, shuffle=True)  
  
  
# 定义神经网络模型  
class SimpleModel(torch.nn.Module):  
    def __init__(self):  
        super().__init__()  
        self.input = torch.nn.Linear(1, 4)  # 隐藏层4个神经元，线性层  
        self.output = torch.nn.Linear(4, 1)  # 输出层，线性层 w1x1+w2x2+b  
    def forward(self, x):  
        y = torch.relu(self.input(x))  
        return self.output(y)  
  
  
model = SimpleModel()  
optimizer = torch.optim.SGD(model.parameters(), lr=0.01)  
criterion = torch.nn.MSELoss()  
writer = SummaryWriter()  
  
# 训练循环  
epochs = 100  
for epoch in range(epochs):  
    for x_batch, y_batch in dataloader:  
        y_predict = model(x_batch)  
        loss = criterion(y_predict, y_batch)  
        optimizer.zero_grad()  
        loss.backward()  
  
        # tensorboard查看梯度分布  
        for name, param in model.named_parameters():  
            writer.add_histogram(name, param.grad, epoch)  
  
        optimizer.step()  
  
# 预测  
# view的参数-1表示自动计算维度，1表示一维向量，将 x_tensor的形状变为 (100, 1)，因为模型的第一层是1*4  
final_prediction = model(x_tensor)  
  
# 绘图  
plt.clf()  
plt.scatter(x, y)  
plt.plot(x, final_prediction.detach().numpy(), color='r')  
plt.show()
```
