```python 定义模型
from torch import nn  
import torch  
  
  
class MnistModel(nn.Module):  
    def __init__(self):  
        super().__init__()  
        self.fc1 = nn.Linear(1 * 28 * 28, 128)  
        self.fc2 = nn.Linear(128, 256)  
        self.fc3 = nn.Linear(256, 128)  
        self.fc4 = nn.Linear(128, 1)  
  
    def forward(self, x):  
        x = torch.relu(self.fc1(x))  # (16, 1*28*28) * (1*28*28, 128) = (16, 128)  
        x = torch.relu(self.fc2(x))  # (16, 128) * (128, 256) = (16, 256)  
        x = torch.relu(self.fc3(x))  # (16, 256) * (256, 128) = (16, 128)  
        x = self.fc4(x)  # (16, 128) * (128, 1) = (16, 1)  
        return x
```

## 代码详解

### 训练循环逐行解释

**初始化部分**
```python
model = MnistModel()  # 创建模型实例
optimizer = torch.optim.SGD(model.parameters(), lr=0.1)  # SGD优化器，学习率0.1
criterion = nn.MSELoss()  # 均方误差损失函数
```

**训练循环**
```python
epochs = 10  # 训练10轮完整数据集
for epoch in range(epochs):  # 外层循环：每个epoch
    for i, (images, labels) in enumerate(train_loader):  # 内层循环：每个批次
        
        # 1. 数据准备
        labels = labels.float()  # 转换为浮点型（MSE要求）
        outputs = model(images.view(16, 1 * 28 * 28))  # 展平：(16,1,28,28) -> (16,784)
        
        # 2. 计算损失
        loss = criterion(outputs, labels.view(16, 1))  # 预测值与真实值的MSE
        
        # 3. 反向传播三步骤
        optimizer.zero_grad()  # 清空上一轮梯度
        loss.backward()        # 计算梯度
        optimizer.step()       # 更新参数
        
        # 4. 日志输出
        if (i + 1) % 100 == 0:
            print(f'Epoch [{epoch+1}/{epochs}], Step [{i+1}/{len(train_loader)}], Loss: {loss.item():.4f}')
```

### 优化版本

```python
import torch
from torch import nn
from torch.utils.data import DataLoader
from torchvision import datasets, transforms

# 数据准备
transform = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize((0.5,), (0.5,))  # 归一化到[-1, 1]
])

train_dataset = datasets.MNIST(root='./data', train=True, transform=transform, download=True)
train_loader = DataLoader(train_dataset, batch_size=64, shuffle=True)

# 模型初始化
model = MnistModel()
device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
model.to(device)

# 优化器和损失函数
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)  # Adam更稳定
criterion = nn.MSELoss()

# 训练循环
epochs = 10
for epoch in range(epochs):
    model.train()
    running_loss = 0.0
    
    for i, (images, labels) in enumerate(train_loader):
        # 移动到GPU
        images = images.to(device)
        labels = labels.to(device).float()
        
        # 动态获取batch_size
        batch_size = images.size(0)
        
        # 前向传播
        outputs = model(images.view(batch_size, -1))  # -1自动计算
        loss = criterion(outputs, labels.view(batch_size, 1))
        
        # 反向传播
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
        
        running_loss += loss.item()
        
        if (i + 1) % 100 == 0:
            avg_loss = running_loss / 100
            print(f'Epoch [{epoch+1}/{epochs}], Step [{i+1}/{len(train_loader)}], Avg Loss: {avg_loss:.4f}')
            running_loss = 0.0

# 保存模型
torch.save(model.state_dict(), 'mnist_model.pth')
```

### 主要改进

1. **动态 batch_size** - 不再硬编码 16，使用 `images.size(0)`
2. **GPU 加速** - 自动检测并使用 CUDA
3. **Adam 优化器** - 比 SGD 收敛更快，学习率 0.001
4. **数据归一化** - 提升训练稳定性
5. **平均损失** - 每 100 步显示平均值，更平滑
6. **模型保存** - 训练后保存权重

### 常见问题

**Q: 为什么要 `optimizer.zero_grad()`？**  
A: PyTorch 默认累积梯度，不清空会导致梯度叠加错误

**Q: MSE 适合分类任务吗？**  
A: 不太适合。分类任务应该用 `nn.CrossEntropyLoss()`

**Q: 学习率 0.1 会不会太大？**  
A: 对 SGD 来说偏大，容易震荡。Adam 的 0.001 更稳定
